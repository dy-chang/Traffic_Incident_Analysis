# Data Note

Raw and processed data files are not included in this repository due to size and access restrictions.

This project uses MDOT CHART event logs (2020–2024) and related transportation datasets for analysis and modeling.

Included in this repository:
- notebooks
- figures
- summary tables
- reproducible workflow documentation

Excluded from this repository:
- raw source data
- large processed parquet files
- institution-specific storage paths
